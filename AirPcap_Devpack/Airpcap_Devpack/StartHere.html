Developer's Pack

This archive contains everything you will need to create wireless capture applications programs using the AirPcap library from CACE Technologies: include files, lib files, dynamic link libraries and driver, online documentation and a set of sample programs.  The sample programs provide a convenient starting point for application development and illustrate the use of all of the AirPcap library functions.

Package Contents

The AirPcap developer's pack consists of the following folders:

  • docs: this folder contains:
    • the AirPcap API online documentation
  • samples: this folder contains a complete set of ready-to-compile sample programs that demonstrate all of the functionality of the AirPcap library.
  • include: .h files needed to compile AirPcap-based programs.
  • lib: .lib files to be linked with AirPcap-based programs.

How to Run the Sample Programs

Consult the readme file in the samples directory.
